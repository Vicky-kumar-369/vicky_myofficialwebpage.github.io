HTML

HTML (Hypertext Markup language) as name suggests it is a markup language used to give structure to web pages. Many people thinks that HTML is a programming language but that's not true, it is a markup language that means we can only give structure to our website i.e. we can enter the texts, check box, tables, text areas , in our website by using it.

Some important terms and points of HTML that we will use further:

  • In HTML, there is opening tags ( < >) and closing tags ( < / >)
  • Tags : It have some attributes or abilities.
    Eg. < input>, < table>, etc
  • Element : Tags and content inside it, is together called Element
    Eg. <p> Lorem ipsum dolor sit amet. </p>
